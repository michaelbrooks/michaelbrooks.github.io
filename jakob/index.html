An Interview with Jakob Nielsen

This application uses natural language processing and information retrieval techniques to simulate a conversation with Jakob Nielsen, based on his Alertbox columns from 1995 to 2011.



Type questions in the box on the left. Press Enter or the Send button to send the message.

If you would like to continue hearing about a topic, try asking a short question like mhm? or simply ?

Note: Does not actually contain Jakob Nielsen.
